Join the conversation

Come hang out on the official Gitter channel. You might also find some of us on the IRC channel at #pakyow on Freenode IRC. Finally, join the mailing list, where early discussions about new Pakyow features often take place.

Contribute to the project

We'd love you to contribute code of documentation. Everything is hosted publicly on Github. Looking through the list of open issues would be a great place to start. If you need more direction, jump on one of the communication channels listed above; we're happy to help!

Report problems you encounter

Found a bug or have an idea for a new feature? Open up an issue on our Github issues page.

Get help from the community

Ask and answer questions on Stack Overflow. They're tagged, so it's easy!
